Redpy Commands

1. avatar [user] - Sends the avatar of the user
2. banner [banner] - Sends the banner of the user

5. 8ball [Question] - Gives you yes, no, maybe responses to your question

-> Utlity Commands

6. Latency - Shows the bot ping rate
